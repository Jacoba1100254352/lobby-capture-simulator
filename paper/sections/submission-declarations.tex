\section*{Data and Code Availability}

The simulator source, scenario configurations, validation scripts, generated paper tables, generated figure inputs, and committed report snapshots are maintained in this repository. Private credentials and raw source payloads are intentionally excluded from version control. The default reproducibility path is \texttt{make test campaign sensitivity ablation interactions source-moments validate calibration-queue paper}; the pinned environmental validation snapshot is generated with \texttt{make snapshot-2024-env}.
